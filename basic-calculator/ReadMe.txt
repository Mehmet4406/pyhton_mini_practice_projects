Operations (Basic Calculator)

A simple Python command-line calculator created as part of my Computer Science coursework.

The program takes two numeric inputs from the user and performs an arithmetic operation based on the selected option.

Features
- Addition
- Subtraction
- Multiplication
- Division
- User input handling via the command line

Concepts Used
- User input and type casting
- Conditional statements (if / elif / else)
- Basic arithmetic operations
