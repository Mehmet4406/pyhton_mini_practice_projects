Conditional Programs

A collection of small Python programs created as part of my Computer Science coursework to practice decision-making using conditional logic and user input.

Programs Included
- Coffee size selector based on user-entered price
- Ice cream size selector based on user-entered price
- User access check using conditional logic

Concepts Used
- Conditional statements (if / elif / else)
- Logical operators
- Nested conditions
- User input and type casting
